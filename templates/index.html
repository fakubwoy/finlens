Have a company Chart of Accounts?
Upload your CoA Excel first, then upload your bank statement — classifications will match your exact GL codes and expense heads. Without it, the system uses a built-in set of ~40 common Indian expense categories.
Your CoA file should have these columns (only account_name is required):
Column Purpose
account_nameExpense head name *required
codeGL / ledger code (e.g. 5001)
llm_keywordsComma-separated keywords for matching
gst_flagGST applicability (Y/N)
tds_sectionTDS section (e.g. 194C, 194J)
is_activeSet to false to exclude a row
